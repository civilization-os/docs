# Spring 框架指南

Spring 是 Java 企业级应用开发的事实标准框架，提供了完整的企业服务编程模型。

---

## 知识大纲

1. **Spring Core (核心机制)**
   - 控制反转 (IoC) 与依赖注入 (DI) 的原理
   - Bean 对象的生命周期与循环依赖解决方案（三级缓存）
   - 面向切面编程 (AOP) 原理与动态代理（JDK Proxy vs CGLIB）
2. **Spring Boot**
   - 自动装配原理与 Starter 机制
   - 外部化配置加载顺序
3. **微服务与 Spring Cloud**
   - 注册中心、配置中心、网关以及负载均衡机制
