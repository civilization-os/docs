# C++ 语言指南

C++ 是一种高效、通用的编程语言，具有面向对象、泛型编程和底层内存操作的特性。

---

## 知识大纲

1. **核心语法基础**
   - 指针、引用与常量限定符（const）
   - 内存模型与管理（Stack vs Heap）
2. **面向对象与现代 C++**
   - 构造函数、析构函数与三/五法则
   - 智能指针（`unique_ptr`, `shared_ptr`, `weak_ptr`）
   - RAII 机制（资源获取即初始化）
3. **标准模板库 (STL)**
   - 容器（`vector`, `list`, `map`, `unordered_map`）
   - 迭代器与算法
