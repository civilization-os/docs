# Java 语言指南

Java 是一种广泛使用的面向对象编程语言，其核心理念是“一次编写，到处运行”（Write Once, Run Anywhere），这得益于 JVM（Java 虚拟机）。

---

## 知识大纲

1. **JVM 核心原理**
   - 内存区域划分（堆、栈、方法区、程序计数器）
   - 垃圾回收机制（GC 算法与垃圾收集器）
   - 类加载器与双亲委派机制
2. **Java 并发编程 (JUC)**
   - 线程池原理与最佳实践
   - 锁机制（Synchronized, ReentrantLock, AQS 原理）
   - 线程安全容器
3. **Java 新特性**
   - 从 Java 8 Lambda 到现代 Java 的演进
