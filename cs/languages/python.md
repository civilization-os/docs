# Python 语言指南

Python 是一种高级、动态的解释型编程语言，以其简洁明了的语法和强大的生态在数据科学、人工智能和 Web 开发中被广泛应用。

---

## 知识大纲

1. **语言高级特性**
   - 装饰器（Decorators）与生成器（Generators）
   - 内存管理与垃圾回收（引用计数、标记清除、分代回收）
   - 全局解释器锁 (GIL) 的原理与多进程/多线程选型
2. **异步编程 (Asyncio)**
   - 协程原理、事件循环与并发控制
3. **生态圈与最佳实践**
   - 虚拟环境管理（Poetry, Conda）
   - 流行框架概览（FastAPI, Django, PyTorch）
