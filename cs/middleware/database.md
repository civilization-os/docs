# 数据库指南

本部分将探讨主流的数据存储与检索技术，包括关系型数据库（RDBMS）与非关系型数据库（NoSQL / NewSQL）。

---

## 知识大纲

1. **MySQL 核心原理**
   - 存储引擎（InnoDB 底层 B+ 树索引结构）
   - 事务特性（ACID）与隔离级别（MVCC 原理）
   - 日志系统（Binlog, Redo log, Undo log）
2. **Redis 缓存技术**
   - 核心数据结构与单线程高性能模型
   - 缓存问题解决方案（击穿、穿透、雪崩）
   - 哨兵与集群（Gossip 协议、哈希槽分片）
3. **分布式存储**
   - 分库分表（分片键设计、跨库查询、分布式 ID 方案）
